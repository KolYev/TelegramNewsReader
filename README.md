## Что делает проект

- Собирает все посты за сегодняшний день из ваших Telegram-каналов
- Сохраняет их в структурированный текстовый файл
- Анализирует новости через локальную языковую модель (без интернета)
- Выдаёт топ-10 хороших и топ-10 плохих новостей дня
- Формирует краткую сводку прошедшего дня

## Стек

`Python` · `Telethon` · `LM Studio / Ollama` · `Qwen / Gemma / Teapot LLM`

## Быстрый старт

```bash
pip install telethon rich requests
python main.py
